What is Galaxy?

Galaxy is a lightweight c# library for servers, built on and supporting .NET Core 5.0. It is unreleased and is currently a simple small project I am building for personal use

Why is Galaxy?

Honestly? I had an issue with ASP.NET Core and took the nuclear route. None of this is really neccessary but i find it fun.

Why not use ASP.NET?

It is hard if not to get ASP.NET to not block without doing some messy thread work. Galaxy solves that by not blocking on the Serer.Start() call, allowing your processes or tests to run freely in the foreground thread while your server is happy in a background thread.
